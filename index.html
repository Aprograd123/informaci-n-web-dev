HTML

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar páginas web en la World Wide Web. Fue desarrollado originalmente por Tim Berners-Lee en 1991 y ha evolucionado a lo largo del tiempo para convertirse en la base de la web moderna.

CSS

CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML (o XML). Mientras que HTML se encarga de la estructura y el contenido de una página web, CSS se encarga de cómo ese contenido debe verse: el diseño, los colores, las fuentes, el espaciado, y más.

JAVASCRIP

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, que se utiliza principalmente para crear páginas web interactivas y dinámicas. A diferencia de HTML y CSS, que se encargan de la estructura y el estilo de una página web, JavaScript permite agregar comportamiento a las páginas, haciendo que puedan responder a las acciones del usuario, interactuar con los servidores, manipular el contenido de manera dinámica, y mucho más.
